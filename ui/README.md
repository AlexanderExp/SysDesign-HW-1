# Демо UI для системы аренды пауэрбанков

Простой веб-интерфейс для демонстрации работы системы на защите.

---

## Быстрый старт

### 1. Запуск системы

```bash
docker compose up -d
```

Демо страница будет доступна, открыв ui/index.html:

```bash
# macOS
open ui/index.html

# Linux
xdg-open demo-ui/index.html

# Windows
start demo-ui/index.html
```
